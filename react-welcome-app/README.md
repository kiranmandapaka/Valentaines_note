# React Welcome App

This is a simple React application that displays a welcome message and allows users to navigate to a thank you page.

## Project Structure

```
react-welcome-app
├── index.html
├── src
│   ├── main.jsx
│   ├── App.jsx
│   ├── pages
│   │   ├── Welcome.jsx
│   │   └── ThankYou.jsx
│   ├── components
│   │   └── SendButton.jsx
│   └── styles
│       └── index.css
├── package.json
├── vite.config.js
├── .gitignore
└── README.md
```

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd react-welcome-app
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to view the application.

## Features

- A welcome page that prompts the user to enter their name.
- A send button that navigates to a thank you page.
- Simple and clean design using CSS for styling.

## Technologies Used

- React
- Vite
- React Router
